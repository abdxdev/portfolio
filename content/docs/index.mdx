description: Interactive documentation for the portfolio API.
---

<Cards>
  <Card title="List GitHub projects" href="/docs/getProjects">
    Returns a curated list of the portfolio owner's GitHub repositories, ordered by priority then creation date.
  </Card>
  <Card title="List skills" href="/docs/getSkills">
    Returns all skill categories with associated metadata such as icon and display colour.
  </Card>
  <Card title="List social links" href="/docs/getSocials">
    Returns all configured social platform links.
  </Card>
  <Card title="List friends" href="/docs/getFriends">
    Returns a list of friends enriched with GitHub profile data and LinkedIn links.
  </Card>
  <Card title="List anime watchlist" href="/docs/getAnimeList">
    Returns the portfolio owner's anime watchlist from AniList.
  </Card>
  <Card title="List game library" href="/docs/getGamesList">
    Returns the portfolio owner's game library from RAWG across all status categories.
  </Card>
</Cards>
